# Security Policy

[日本語版 / Japanese Version](SECURITY.ja.md)

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via GitHub Issues.

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | ✅        |

## Security Practices

- Only demo credentials for test sites are used
- No production secrets are included
- Only public APIs are used (JSONPlaceholder, Countries GraphQL)
